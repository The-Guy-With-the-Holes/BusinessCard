  • Download Tails Usb Driver and Balena etcher
  • Restart your Pc in the grub(bios) menu. This can be accomplished via pressing f10* during the boot sequence
    (the key to enter bios normally ranges from f10-f12. this may vary depending on the manufacturer of your pc.)
  • In the Grub menu navigate to settings > boot order using the arrow keys rearrange the order so that USB is the top priority and confirm/save it(This will allow your pc to load an OS from the attached USB)
  • Once the boot order change is made and saved, Plug the foramtted USB drive into your personal computer and restart it. if done correctly, the pc should initialise from the USB you have just plugged in and will load a grub menu with some different Tails OS you can run!
    (If this did not work, confirm that your boot load order is correct)
  • Once the new tails OS is loaded, you are safe to plug in the raspberrypi pico
  • Download and install Thonny IDE on your computer if you didn't download the tails OS with Thonny prebuilt (The one on this page).
  • Open Thonny IDE and navigate to configure interpreter setting in the run tab
  • In the configuration setting, set the interpreter to "MicroPython(generic)" and set the port to "Board in FS mode"
  • Navigate back to the Thonny main page and you are ready to start coding, we recommend going to the view tab and making sure "files" is selected, as this will give you a tree overview of files
  • The bottom area of the Thonny main page should have a section called "shell" this is an interactive python area press ctrl+c to stop the boards functionality
  • Now that the board is no longer executing python, you should see in the tree tab that the rapberry pi and it's files are editable, open the file "creds.py" in the IDE, this is the file that will hold all your passwords.

    Hardware Setup:

    • Connect the Pimoroni display pack to your Raspberry Pi Pico following the manufacturer's instructions.
    • Ensure that your Raspberry Pi Pico is powered and connected to your computer via USB.

    Writing the Code:

    • Write the code for your password manager using Python. This code will handle storing, retrieving, and managing passwords securely.
    • Make sure to include functions for adding, editing, and deleting passwords, as well as displaying them on the Pimoroni display.

    Running the Code:

    • Copy the code to your Raspberry Pi Pico using Thonny IDE. Make sure to save the code on the device.
    • Run the code on the Raspberry Pi Pico by pressing the "Run" button in Thonny IDE.

    Connecting to the Device:

    • After running the code, you may encounter an infinite while loop that prevents easy access to the device.
    • To connect to the device and interact with it, follow these steps:
      1. Connect the Raspberry Pi Pico to your computer via USB.
      2. Open Thonny IDE on your computer.
      3. Press and hold the "Ctrl" key on your keyboard.
      4. While holding the "Ctrl" key, click the "Stop" button in Thonny IDE to interrupt the infinite while loop.
      5. You should now be able to access the device and interact with your password manager.

    Using the Password Manager:

    • Once connected, use the Pimoroni display to navigate through the options provided by your password manager.
    • Follow the on-screen instructions to add, edit, delete, and retrieve passwords securely.

    Safety Precautions:

    • Always ensure that your password manager code is secure and does not expose sensitive information.
    • Keep your Raspberry Pi Pico and Pimoroni display pack in a safe and secure location to prevent unauthorized access.
